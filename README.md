# 🦞 ClawRecall - Free Version

**Permanent memory upgrade for OpenClaw agents** — Never forget again.

## One-Command Install (Windows, Mac & Linux)

```bash
curl -fsSL https://raw.githubusercontent.com/clawrecall/clawrecall/main/install-clawrecall.sh | bash
```

**Windows users** (PowerShell):
```powershell
powershell -ExecutionPolicy Bypass -File (New-Object System.Net.WebClient).DownloadString('https://raw.githubusercontent.com/clawrecall/clawrecall/main/install-clawrecall.ps1')
```

## After Install
1. Paste the block shown in terminal into your OpenClaw agent  
2. Say: “Remember this forever: Test”  
3. Open the dashboard in browser
